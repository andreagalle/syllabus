\section{Grading}

Your grade will come from the following sources:

\begin{itemize}
\itemsep1pt\parskip0pt\parsep0pt
\item
  Discussion: 35\%
\item
  Short Papers: 20\%
\item
  Blackboard Posts: 15\%
\item
  Final Project: 30\%
\end{itemize}

The end of this syllabus includes a longer description of what sort of
work will receive an ``A,'' a ``B,'' and so forth.
